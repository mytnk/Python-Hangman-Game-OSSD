# Hangman Game

Simple Hangman game written in Python.

## Requirements

- Python 3.8 or higher

Check version:

python --version


## Start the Game

```python
python "Hangman Program"/hangman.py
```

If python does not work:

```python
python3 "Hangman Program"/hangman.py
```
